enum PolKitErrorCode

typedef enum
{      
        POLKIT_ERROR_OUT_OF_MEMORY,
        POLKIT_ERROR_POLICY_FILE_INVALID
} PolKitErrorCode;

Error codes returned by PolicyKit

POLKIT_ERROR_OUT_OF_MEMORY

 Out of memory

POLKIT_ERROR_POLICY_FILE_INVALID

 There was an error parsing the given policy file
